# Referenced
This project was referenced from https://www.yalco.kr/lectures/graphql-apollo/

# Preparations
1. node js
2. vscode or editor
3. terminal

# Install
1. run `npm install`

# Run Server
1. run `nodemon start` or `sudo nodemon start`

# Run Query
1. enter `http://localhost:4000/`
2. type and run
```graphql
query{
  equipments {
    id
  }
}
```
